## Features
-   **Dynamic Introduction**: A welcoming home section with a typing animation that cycles through different roles like "Web Developer," "Web Designer," and "Software Developer."
-   **Categorized Skills**: A well-organized section displaying technical skills in Frontend, Backend, and Tools.
-   **Responsive Design**: The layout adapts to various screen sizes, ensuring a seamless experience on both mobile and desktop devices.
-   **Interactive UI**: Smooth hover effects on social media icons, buttons, and navigation links to enhance user engagement.
-   **Social Media Integration**: Quick links to my professional and social profiles on LinkedIn, GitHub, Twitter (X), and Instagram.

## Technologies Used
-   **HTML5**: For the structure and content of the website.
-   **CSS3**: For styling, layout, animations, and responsive design.
-   **Font Awesome**: For scalable vector icons.

2.  **Navigate to the project directory:**
    ```bash

    ```
